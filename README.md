# dotfiles

## Install by dot

```
git clone https://github.com/sus-happy/dotfiles.git .dotfiles
cd ~/.dotfiles
./install.sh
```

## Install by [homebrew](https://brew.sh/)

```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew bundle --global
```
